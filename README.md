# Java-Basics

6 projects, as listed below, will be create:

1. Simple Calculator - Performs basic arithmetic operations (add, subtract, multiply, divide)
2. Number Guessing Game - Interactive game where you guess a random number between 1-100
3. Student Grade Manager - Manages student grades, calculates averages, and assigns letter grades
4. To-Do List - Add, remove, and display tasks using ArrayList
5. Temperature Converter - Converts between Celsius and Fahrenheit
6. Bank Account System - Simulates basic banking operations with deposits and withdrawals
